GitHub 文件加速

GitHub 文件链接带不带协议头都可以，支持 Release、Archive 以及 Raw ，右键复制出来的链接都是符合标准的，更多用法、Clone 加速请参考 这篇文章。

Release、Archive 以及 Raw 等均使用 Replit 加速。注意！不支持项目文件夹。

因为 Replit 免费版限制，若长时间未活动，则会进入休眠状态，当再次有 Http 请求时，需要最长 1 分钟的唤醒时间，此时请等待。

如果想加速本站不支持的的网页链接或下载链接，你可以使用 https://rp.zkitefly.eu.org/ + [你要加速的链接] 的组合访问链接。

例如 https://rp.zkitefly.eu.org/https://google.com 、https://rp.zkitefly.eu.org/https://speed.hetzner.de/1GB.bin
